Política de Privacidade

Última atualização: 22 de outubro de 2025

Domínio: https://firecode.site

URL: https://firecode.site/politica-de-privacidade

Responsável: FireCode Tecnologia e Automação

⸻

1. Introdução

Esta Política de Privacidade explica como tratamos as informações coletadas por meio do Agente I.A FireCode, um sistema de automação que opera via WhatsApp, Instagram, Facebook e outras plataformas integradas, utilizando as APIs oficiais da Meta Platforms Technologies, Inc.

Ao utilizar nossos serviços, você concorda com esta Política e autoriza o tratamento de dados conforme a Lei Geral de Proteção de Dados (Lei nº 13.709/2018 - LGPD).

⸻

2. Dados Coletados

Durante a interação com o Agente I.A ou serviços FireCode, podemos coletar:

  • Dados de identificação: nome, número de telefone, e-mail (se fornecido), ID de usuário da Meta.
  • Mensagens e interações: textos enviados ao Agente I.A via WhatsApp ou redes sociais.
  • Dados técnicos: logs de uso, data e hora de mensagens, idioma, tipo de dispositivo e informações de conexão.
  • Dados fornecidos voluntariamente: informações para agendamento, lembretes, suporte, cadastro ou execução de serviços.

🔒 Não coletamos dados sensíveis (como financeiros, médicos ou biométricos) nem acessamos mensagens fora das conversas com o Agente I.A.

⸻

3. Finalidade do Uso dos Dados

Os dados coletados são utilizados para:

  • Operar o Agente I.A FireCode e responder às solicitações dos usuários;
  • Agendar lembretes, compromissos ou eventos solicitados;
  • Registrar interações e melhorar a precisão da inteligência artificial;
  • Enviar comunicações úteis, notificações e confirmações automáticas;
  • Cumprir obrigações legais e políticas das plataformas integradas (Meta, Google, etc.).
⸻

4. Compartilhamento de Dados

Os dados poderão ser compartilhados apenas com:

  • Meta Platforms Technologies, Inc., provedora do WhatsApp Cloud API e Instagram Graph API;
  • Serviços de nuvem e automação (Google Cloud, n8n, AWS, Render, etc.), quando necessários para o funcionamento;
  • Autoridades legais, mediante solicitação formal.

Os dados não são vendidos, alugados ou usados para fins publicitários.

⸻

5. Retenção e Exclusão dos Dados

Os dados são armazenados pelo tempo necessário para executar o serviço solicitado.

O usuário pode solicitar a exclusão de seus dados a qualquer momento, enviando um e-mail para:

📩 privacidade@firecode.site

A exclusão será processada em até 30 dias úteis, com remoção completa dos registros associados.

⸻

6. Direitos do Usuário (LGPD)

O usuário tem direito de:

  • Acessar, corrigir ou excluir seus dados pessoais;
  • Revogar consentimento;
  • Solicitar portabilidade dos dados;
  • Saber com quem seus dados são compartilhados;
  • Solicitar informações sobre a finalidade do tratamento.

As solicitações podem ser feitas através do e-mail: privacidade@firecode.site

⸻

7. Segurança dos Dados

A FireCode adota medidas técnicas e organizacionais para proteger os dados pessoais contra acesso não autorizado, uso indevido ou perda acidental.

As mensagens trafegam via criptografia ponta a ponta (WhatsApp Cloud API) e os dados armazenados seguem padrões internacionais de segurança.

⸻

8. Base Legal

O tratamento de dados é realizado com base nas seguintes hipóteses da LGPD:

  • Consentimento do titular (art. 7º, I);
  • Execução de contrato ou procedimento preliminar (art. 7º, V);
  • Cumprimento de obrigação legal/regulatória (art. 7º, II);
  • Legítimo interesse do controlador (art. 7º, IX).
⸻

9. Alterações nesta Política

Esta Política poderá ser atualizada a qualquer momento.

A versão mais recente estará sempre disponível em:

🔗 https://firecode.site/politica-de-privacidade

O uso contínuo dos serviços após mudanças significa concordância com os novos termos.

⸻

Para dúvidas, solicitações ou reclamações sobre privacidade e proteção de dados:
